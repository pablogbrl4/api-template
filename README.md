Objetivo:

	Este modelo de projeto tem como intuito de apresentar uma estrutura padrão para projeto onde o Domínio "Regras de Negócios sejam muito complexas e extensas" seja muito importante.

Pastas:
	
	src:
		Toda a estrutura do projeto ficará nesta pasta. 

	test:
		Todos os testes relacionadas as camadas ficarão nesta pasta.

	build:
		Todo o template para publicar o projeto ficará nesta pasta.

	doc:
		Documentação do projeto


OBS: Todas as pastas estão em Beta.
